A implemetação se trata de um projeto feito na IDE codeBlocks.
Instalando o codeblocks utilize a opção "abrir projeto" e seleciona o arquivo "AGRMExato.cbp", em sequida utiliza o botão da engrenagem amarela para compilar e o botão de play para rodar o projeto.

Para compilar sem a instalação da IDE basta utilizar o seguinte comando a partir deste diretório:
g++ main.cpp src/* -I include -o exe

e para executar:
./exe
